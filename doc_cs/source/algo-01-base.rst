算法
===============



算法，抽象地讲就是指一系列操作，是我们解决问题的方法。
如何评估算法的优劣呢？我们一般通过两个尺度来衡量：时间复杂度和空间复杂度。

首先，算法是能解决问题的。同时算法要求必须有一定的输入。

总结下来就成为了以下几点：

- 0个或多个输入，但一般有输出
- 确定性
- 可行性
- 健壮性

时间复杂度
----------

时间复杂度，是计算机在一定的运算场景下，消耗的时间长短

空间复杂度
----------

指的是在一定时间内，所占用的空间大小。一般是内存大小。
有以下几种：

- 常数级
- 线性级
- 对数级
- 线性对数级
- 平方级
- 立方级
- 阶乘级等等
